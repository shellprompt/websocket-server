# Websocket Echo Server
A simple websocket server written in Rust that echoes back any text message it receives.

## Features
- Simple echo (text messages only)
- Supports multiple simultaneous clients

## Requirements
- Rust 1.70+

## Installation

```bash
# Clone the repo
git clone https://github.com/shellprompt/websocket-server.git
cd websocket-server
cargo build # or cargo run if you wish to instantly launch. 
```
